# Microsoft for Startups

Microsoft for Startups bundles Azure cloud credits with GitHub Enterprise, Microsoft 365, and go-to-market support. For startups building on the Microsoft stack — or selling into enterprise Microsoft environments — it is often the most practical of the big three cloud startup programs.

## What you get

- **Azure credits.** Up to $150,000 USD in Azure credits for qualified startups.
- **GitHub Enterprise.** Free GitHub Enterprise licenses for up to 20 seats.
- **Microsoft 365.** Productivity tools for the founding team.
- **Technical support.** Architecture guidance and access to Microsoft engineers.
- **Go-to-market support.** Marketplace listing, co-sell opportunities, and customer introductions.
- **AI and Copilot benefits.** Access to Azure OpenAI Service, GitHub Copilot, and Microsoft AI resources.

## Eligibility

- Startups that are privately held, less than 7 years old, and have raised less than $10 million in funding.
- Must be building a software-based product or service.
- Some benefits require partnership with a Microsoft for Startups partner organization.

## How to apply

1. Visit [Microsoft for Startups](https://www.microsoft.com/en-us/startups).
2. Complete the application with your company and product details.
3. If approved, activate Azure, GitHub, and Microsoft 365 benefits.
4. Work with a Microsoft partner or account contact for higher-tier benefits.

## Caveats

- SMF Clearinghouse does not administer this offer.
- Eligibility and credit amounts change; verify directly with Microsoft.
- Credits are Azure-specific and expire if not used within the program window.
- GitHub Enterprise and Microsoft 365 benefits have their own activation requirements.

## Best for

- Startups building on .NET, Azure, GitHub, or Microsoft 365.
- B2B startups selling into enterprises that already use Microsoft.
- Teams that want GitHub Copilot and Azure OpenAI Service included in their startup package.

## Verdict

Microsoft for Startups is the best fit for teams already in the Microsoft ecosystem. The Azure credit amount is lower than AWS Activate's top tier, but the inclusion of GitHub Enterprise and Microsoft 365 can make the total package more valuable for the right team.
