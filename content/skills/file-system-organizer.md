# File System Organizer

Clean up messy directories with natural-language rules. This skill renames files, moves them into folders, removes duplicates, and creates archives based on date, type, or content patterns.

## What it is

A file manipulation skill that lets you describe what you want organized and where. It previews changes before applying them so you do not accidentally lose data.

## Who it targets

- Users with years of accumulated Downloads and Desktop clutter.
- Photographers, designers, and researchers managing asset folders.
- Teams standardizing project directory layouts.

## Dependencies

- Hermes Agent
- Local or mounted directory access
- Optional - trash/recovery tool for safety

## How to install

```bash
hermes skill install file-system-organizer
```
