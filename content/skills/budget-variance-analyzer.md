# Budget Variance Analyzer

Compare actual vs budgeted spending and explain variances.

## What it is

This skill gives your agent a structured way to handle budget variance analyzer tasks. It wraps the necessary tools, prompts, and output formatting into a reusable command you can invoke from chat, cron, or a messaging gateway.

## Who it targets

- Teams and individuals using **Hermes Agent** who want to automate repetitive work.
- Users looking for a starting point they can customize for their own stack.
- Anyone who prefers natural-language commands over manual tool configuration.

## Dependencies

- Hermes Agent
- Budget and actuals CSV/sheet

## How to install

```bash
hermes skill install budget-variance-analyzer
```

Or install through the Hermes Desktop skills hub.
