# NVIDIA Changelog

## 2026-06

### Nemotron 3 Ultra
Released June 4, 2026. Nemotron 3 Ultra is a 550B-parameter Mixture-of-Experts model with 55B active parameters, optimized for orchestrating complex, long-running agents. NVIDIA is positioning it as a reasoning and planning model for enterprise agentic workflows, not just chat.

## 2026-05

### NVIDIA Agent Toolkit
Announced May 31, 2026. The Agent Toolkit bundles:
- **NeMoClaw blueprints:** reference agent architectures for common enterprise tasks.
- **Nemotron models:** reasoning and reward models for agent training and evaluation.
- **OpenShell secure runtime:** a hardened runtime for agent execution.
- **CUDA-X libraries:** optimized primitives for retrieval, embedding, and inference.

### Enterprise software partnerships
NVIDIA announced that major enterprise software vendors are building agents on top of NVIDIA's stack. The pitch is that NVIDIA can serve as the infrastructure layer for agentic AI in regulated industries.

## 2026-03

### Open model family expansion
NVIDIA expanded its open model families for agentic, physical, and healthcare AI. Nemotron 3 Omni models were added for multi-modal understanding tasks.

## What this means for users

NVIDIA is moving up the stack. It no longer wants to be just the GPU provider — it wants to own the agent runtime, model, and reference architecture. For teams already on NVIDIA hardware, NIM and NeMoClaw lower the barrier to self-hosted agents. For everyone else, the value proposition depends on whether you are willing to deepen your NVIDIA dependency.

## What to watch

- Whether Nemotron 3 Ultra gets independent benchmark validation.
- Pricing and availability of NIM microservices for smaller teams.
- How NeMoClaw compares to open-agent frameworks like OpenClaw, LangGraph, and AutoGen.
