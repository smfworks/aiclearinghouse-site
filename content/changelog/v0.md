# v0 Changelog

## Recent highlights

### Design-system-aware generation
- v0 can now generate components that match an existing design system or component library, making handoff to production code smoother.
- Improved React and Tailwind output quality, with fewer broken layouts and better accessibility defaults.

### Next.js integration
- Better support for Next.js App Router patterns.
- One-click export to a GitHub repository or Vercel project, including environment setup.

## What this means for users

v0 remains the fastest way to go from a text description to a polished React component. The design-system awareness is a meaningful step toward production-ready output, though you will still need to wire up backend logic yourself.

## What to watch

Vercel is likely to deepen v0's integration with its deployment platform. If you are already on Vercel, the one-click deploy flow is a real time-saver.
